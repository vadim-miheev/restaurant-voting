# Restaurant voting
Voting system (Backend) for deciding where to have lunch


## Stack
- [JDK 17](http://jdk.java.net/17/)
- Maven
- Spring Boot 2.7
- Spring MVC
- Spring Data JPA (Hibernate)
- Spring Security
- Lombok
- Jackson
- JUnit 5
- Swagger/OpenAPI 3.0
- H2

## Run
Execute in root directory:
```
mvn spring-boot:run
```
